module plug ทำงานเกี่ยวกับ comment.
comment_after_newcomment 	เรียกทำงานหลังเพิ่ม comment สำเร็จแล้ว โดยส่งค่าต่างๆไปเป็น array
comment_after_updatecomment	เรียกทำงานหลังแก้ไข comment สำเร็จแล้ว โดยส่งค่าไปเป็น array
comment_after_delete		เรียกทำงานหลังลบ comment แล้ว	ส่งค่าเป็น comment_id (integer)
comment_spam_check		เรียกตรวจ spam โดยส่งค่าต่างๆไปเป็น array รวมทั้งarray permalink_url. การส่งค่ากลับ หากไม่พบ spam ต้องส่งค่ามาเป็น normal
comment_modifybody_value		เรียกทำงานแก้ไขเนื้อหาความคิดเห็น	ส่งค่าไปเป็น string.
comment_admin_index_top		เรียกทำงานในหน้า comment admin ด้านบน ต่อจากตรงที่นับความคิดเห็นประเภทต่างๆ. 
comment_admin_index_options	เรียกทำงานในหน้า comment admin ด้านล่างใน select options โดยให้ return ส่งค่า <option> ต่างๆมา
comment_admin_process		เรียกทำงานในหน้า process หลังจากที่ตรวจแล้วไม่พบค่า act ในระบบหลัก โดยส่งค่าเป็น array ไป และไม่ต้องรับค่าใดๆกลับมา
